Preguntas

Describe el archivo package.json.:

"name:" describe el nombre del proyecto, que es "servidor".
"version" es la version del proyecto que es "1.0.0".
"description" es la descripcion del proyecto, que esta vacia porque es opcional y la introduce el creador.
"main" es el archivo principal del proyecto, que es "server.js".
"scripts" son los scripts de nmp que pueden ejecutarse, en este proyecto se utiliza "start" para ejecutar el archivo con node y "dev" para ejecutar el servidor con nodemon.
"author" es el autor del proyecto que es "Carlos Galicia".
"licence" es la licencia bajo la cual se distribuye el proyecto, que es "ISC".
"devDependencies" son las dependencias bajo las cuales el proyecto se desarrolla, la cual es "nodemon" version 3.1.0.
"dependencies" son las dependencias que el proyecto utiliza para ejecutarse, que son "body-parser", "date-fns", "root", y "uuid" y sus respectivas versiones.
