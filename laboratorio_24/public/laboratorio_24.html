Preguntas

¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications)?:

Permite que las aplicaciones web respondan a las acciones del usuario sin necesidad de recargar la página completa, tambien permiten actualizar solo partes específicas de una página web en lugar de recargarla completamente, lo que reduce el tiempo de espera del usuario y la cantidad de datos que se deben de renderizar.

¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?:

AJAX puede exponer datos sensibles en el tráfico de la red si no se usa HTTPS. Las validaciones tambien deben de hacerse del lado de servidor, debido a que si solo se hacen del lado del cliente, los atacantes pueden eludirlas manipulando las solicitudes AJAX directamente.

¿Qué es JSON?:

Es un formato para el intercambio de datos, el cual es fácil de interpretar y generar para las computadoras. El JSON se utiliza en aplicaciones web para enviar datos entre servidor y cliente.
