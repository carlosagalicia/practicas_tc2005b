Preguntas

¿Qué diferencias y semejanzas hay entre Java y JavaScript?:

En cuanto a sus similitudes, ambos lenguajes comparten una sintaxis similar a C, pueden ser interpretados o compilados "just in time", pueden ejecutarse en una variedad de sistemas operativos y los dos son utilizados en el desarrollo web. Las diferencias que tienen son que Java esta orientado a objetos de propósito general donde se ejecuta tanto del lado del cliente como del lado del servidor, mientras que JavaScript es principalmente utilizado para agregar interactividad a las páginas web, donde solo se ejecuta del lado del cliente. Tambien en JavaScript el tipado es dinamico, mientras que en Java los tipos de datos se declaran explicitamente.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*):

El objeto date tiene metodos como "getDate()", "getMonth()", "getFullYear()", "getHours()", "getMinutes()", etc.

¿Qué métodos tienen los arreglos? (Menciona al menos 5*):

Los arreglos tiene metodos como "push()", "pop()", "splice()", "slice()", "indexOf()", etc.

¿Cómo se declara una variable con alcance local dentro de una función?:

Se puede declarar utilizando las palabras "var", "let" o "const" adentro de la función, dependiendo si la variable es mutable o no.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?:

Se pueden generar colisiones de nombres con variables locales de otras funciones
